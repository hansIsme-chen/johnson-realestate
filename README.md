## 🚀 **Project Overview**

這是一個高效能的 **房地產交易平台**，使用 **Python** 與 **Django** 框架開發。本專案的核心目標是透過 **AI 輔助開發（Vibe Coding）**，在極短的時間內實現從資料庫設計到前端視覺化的完整流程。

---

## 🛠 **Key Features (核心功能)**

* **AI-Enhanced Development**: 全面導入 **Claude** 與 AI 寫碼工具，優化複雜的後端邏輯並加速 Bug 排查。
* **Interactive Map Integration**: 成功串接 **Google Maps API**，提供用戶直觀的房產地理位置資訊。
* **Dynamic Search & Filtering**: 建立強大的**搜尋與篩選系統**，包含地區、價格與細節頁面跳轉。
* **Optimized Data Architecture**: 運用**高等資料庫**知識設計正規化 Schema，並妥善管理 **Database Migrations**。
* **Containerized Environment**: 整合 **Docker** 技術，確保開發環境的一致性並簡化部署流程。

---

## 💻 **Tech Stack (技術棧)**

* **Back-end**: **Python**, **Django**。
* **Cloud & DevOps**: **Google Cloud Platform (GCP)**, **Docker**。
* **Database**: SQLite / PostgreSQL (經由 Django ORM 管理)。
* **AI Tools**: **Claude**, **Gemini**, GitHub Copilot。
